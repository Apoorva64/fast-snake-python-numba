## About The Project

The projects aims to provide a fast implementation of the snake game in python to be used for machine learning.
The game is rendered with pygame, but the game logic is implemented in python code which is supercharged with numba.
This allows for a very fast game logic, which is important for machine learning
(ex: run lots of games in a short time to speed up training).


<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Built With

* [Python](https://www.python.org/)
* [Pygame](https://www.pygame.org/)
* [Numba](https://numba.pydata.org/)
* [Numpy](https://numpy.org/)

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Apoorva64/fast-snake-python-numba.git
   ```
2. Install python packages with pip
   ```sh
   pip install -r requirements.txt
   ```

## Usage

A simple example of how to use the game is provided in the `main.py` file.
With the installation done, you can run a simple version of the game with

```sh
python main.py
```
